Watch the rollout:

  kubectl --namespace {{ .Release.Namespace }} rollout status {{ .Values.kind | lower }}/{{ include "ferro-cargo-registry-server.fullname" . }}

Tail the logs:

  kubectl --namespace {{ .Release.Namespace }} logs -l \
      app.kubernetes.io/instance={{ .Release.Name }} -f --tail=50

Hit the registry (port-forward from your shell):

  kubectl --namespace {{ .Release.Namespace }} port-forward svc/{{ include "ferro-cargo-registry-server.fullname" . }} 8081:{{ .Values.service.port }}

  curl -s http://localhost:8081/live          # liveness (200 OK)
  curl -s http://localhost:8081/ready         # readiness (200 OK)
  curl -s http://localhost:8081/healthz       # {"status":"ok"}
  curl -s http://localhost:8081/config.json   # cargo sparse-index config

Point cargo at it (~/.cargo/config.toml):

  [registries.ferro]
  index = "sparse+http://localhost:8081/index/"
